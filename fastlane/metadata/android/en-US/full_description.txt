<p><p dir="auto">A Small App for Enhanced Privacy &amp; Security</p><p dir="auto"><strong>Memory Guardian</strong> is a lightweight Android app that enhances user privacy and security by automatically clearing the clipboard at set intervals. This helps protect sensitive information, like passwords, from potential leaks to other applications.</p><p dir="auto"><strong>Key Features:</strong></p><ul dir="auto"><li><strong>Automatic Clearing:</strong> Set customizable intervals for automatic clipboard clearing.</li><li><strong>Manual Clearing:</strong> Use the "Clear" button for immediate action.</li><li><strong>Clipboard Editing:</strong> Edit clipboard content directly within the app for quick adjustments.</li><li><strong>Notifications:</strong> Get notified when the clipboard is cleared.</li><li><strong>Dark Mode:</strong> Switch between light and dark themes for a personalized experience.</li><li><strong>Tiny:</strong> The app is small in size and does not consume much storage space.</li><li><strong>No Permissions:</strong> Memory Guardian does not require any permissions to function, ensuring user privacy.</li><li><strong>No Ads:</strong> Enjoy an ad-free experience with no interruptions.</li><li><strong>Minimalistic:</strong> Experience a clean and clutter-free interface for a distraction-free user experience.</li><li><strong>Open Source:</strong> Contribute to the project and help improve the app for everyone.</li></ul></p>
